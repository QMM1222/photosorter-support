技术支持

常见问题解答

基本问题

问：PhotoSorter是否支持离线使用？

答：是的，PhotoSorter完全支持离线使用。应用所有功能都在本地处理，不需要网络连接。

问：PhotoSorter需要访问我的照片库权限吗？

答：是的，应用需要照片库访问权限才能管理您的照片。我们只请求必要的权限，并保证所有照片处理都在本地完成，不会上传到任何服务器。

问：如何授予照片库访问权限？

答：首次启动应用时，系统会自动请求权限。如果您不小心拒绝了，可以在设备的"设置">"隐私">"照片"中找到PhotoSorter，并开启权限。

问：使用PhotoSorter是否会影响原始照片？

答：不会。PhotoSorter不会修改您的原始照片，而是通过创建相簿和分类来帮助您整理照片。原始照片保持不变。

使用指南

快速入门

  • 授予权限 - 首次启动应用时，允许照片库访问权限
  • 浏览照片 - 在"未整理"和"全部"标签之间切换，浏览您的照片库
  • 创建相簿 - 点击相簿列表最右侧的"新建相簿"，输入名称创建自定义相簿
  • 添加照片到相簿 - 选中照片后，点击底部相簿名称将照片添加至相簿

高级技巧

  • 批量整理技巧 - 使用"全选"按钮快速选择当前页面所有照片
  • 网格布局切换 - 根据需求切换不同网格布局，浏览大量照片时选择9格布局，查看细节时使用单张布局
  • 照片排序 - 所有照片默认按拍摄时间从新到旧排序，便于找到最近拍摄的照片

Technical Support

Frequently Asked Questions

Basic Questions

Q: Does PhotoSorter work offline?

A: Yes, PhotoSorter fully supports offline use. All app functionality is processed locally and doesn't require an internet connection.

Q: Does PhotoSorter need access to my photo library?

A: Yes, the app requires photo library access permission to manage your photos. We only request necessary permissions and guarantee that all photo processing is done locally, without uploading to any servers.

Q: How do I grant photo library access permission?

A: When you first launch the app, the system will automatically request permission. If you accidentally denied it, you can find PhotoSorter in your device's "Settings" > "Privacy" > "Photos" and enable the permission.

Q: Will using PhotoSorter affect my original photos?

A: No. PhotoSorter doesn't modify your original photos but helps you organize them by creating albums and categories. The original photos remain unchanged.

User Guide

Quick Start

  • Grant Permission - Allow photo library access when first launching the app
  • Browse Photos - Switch between "Unsorted" and "All" tabs to browse your photo library
  • Create Albums - Click "New Album" on the far right of the album list and enter a name to create a custom album
  • Add Photos to Albums - Select photos and click an album name at the bottom to add photos to that album

Advanced Tips

  • Batch Organization - Use the "Select All" button to quickly select all photos on the current page
  • Grid Layout Switch - Switch between different grid layouts based on your needs; choose 9-grid layout when browsing many photos, use single photo layout when checking details
  • Photo Sorting - All photos are sorted by capture time from newest to oldest by default, making it easy to find recently taken photos
